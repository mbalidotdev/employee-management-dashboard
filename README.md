# Employment Management Dashboard

A professional HTML tables project showcasing advanced table features and semantic markup for displaying employee data, payroll information, performance metrics, and benefits enrollment.

I built this project to strengthen my understanding of HTML tables.

---

## Features

### 1. Company Overview Dashboard

- Key performance metrics
- Year-over-year comparisons
- Status indicators
- Semantic table headers with proper scope attributes

### 2. Department Directory

- **Rowspan implementation** for department grouping
- Complete employee contact information
- Organized by department hierarchy
- Column styling with `<colgroup>`

### 3. Payroll Report

- **Complex colspan and rowspan combinations**
- Department-wise salary breakdown
- Bonus calculations
- Subtotals and grand totals
- Confidential data handling

### 4. Performance Reviews

- **Multi-level headers** using colspan and rowspan
- Performance metrics by category
- Rating system implementation
- Department averages

### 5. Benefits Enrollment

- Enrollment status tracking
- Multiple benefit categories
- Enrollment rate calculations
- Footer summary statistics

### 6. Document Metadata

- Report information table
- Proper document classification
- Audit trail information
